# **App Name**: Franchise Flow

## Core Features:

- Billing Form: Input fields for capturing customer details, service description, and billing amount.
- Billing Summary: Display a summary of the billing information for review.
- Invoice Number Generation: Unique invoice number for each billing record.
- Invoice History: Display past invoices.
- Suggested Products: Based on invoice history, this tool suggests products and/or services the customer might be interested in, to include on their next invoice.

## Style Guidelines:

- Primary color: Neutral white or light gray for a clean look.
- Secondary color: Light blue for accents and interactive elements.
- Accent: Teal (#008080) for highlighting important information.
- Clean and well-organized layout for easy navigation.
- Clear and readable font for form labels and data display.
- Subtle animations for form transitions and feedback.
